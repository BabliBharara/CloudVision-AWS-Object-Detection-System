# CloudVision – AWS Object Detection System

CloudVision is a serverless object detection web application built using AWS Rekognition, AWS Lambda, Amazon API Gateway, and Amazon S3.

## Features

* Upload images through a web interface
* AI-powered object detection using AWS Rekognition
* Confidence score display
* Bounding box visualization
* Serverless architecture

## Technologies Used

* AWS Rekognition
* AWS Lambda
* Amazon API Gateway
* Amazon S3
* Python
* Boto3
* HTML
* CSS
* JavaScript

## Architecture

Frontend (HTML/CSS/JavaScript)
→ API Gateway
→ AWS Lambda
→ AWS Rekognition
→ Detection Results
